---

title: Sobre esta web
slug: sobre
-----------

Este sitio web ofrece información sobre mantenimiento, reparación y recambios para aspiradores domésticos.

El objetivo es ayudar a identificar piezas compatibles, resolver problemas habituales y facilitar la búsqueda de repuestos.

La información se basa en documentación de fabricantes, experiencias de usuarios y análisis de compatibilidad entre modelos.

Algunos enlaces presentes en la web son enlaces de afiliado.
